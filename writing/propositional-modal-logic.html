Semantic Entailment ⊨

The central relation of propositional logic. What it means, and its key properties.

Definition

The core semantic relation
Γ ⊨ A

Every valuation v that makes all sentences in Γ true also makes A true. In other words: there is no possible assignment of truth values that satisfies Γ but falsifies A.

Key Properties of ⊨

Reflexivity
A ⊨ A
Any sentence entails itself.
Monotonicity
If Γ ⊨ A, then Γ ∪ Δ ⊨ A
Adding more premises never breaks an entailment.
Transitivity (Cut)
If Γ ⊨ A and Δ∪{A} ⊨ B, then Γ∪Δ ⊨ B
Chain entailments together. A is the bridge.
Compactness
If Γ ⊨ A, some finite Γ₀ ⊆ Γ already suffices
You never need infinitely many premises.

Key: A doesn't need to be in Γ — it's entailed by Γ. Γ produces A silently. Δ uses A to produce B. Pooling Γ ∪ Δ gives you B directly.

Tautology

⊨ A (empty premise set entails A)

True under every valuation. No premises needed. Examples:

A ∨ ¬A
A → A
(A → B) → ((B → C) → (A → C))

Contradiction

Unsatisfiable — false under every valuation

False under every valuation. A contradiction entails everything (ex falso):

A ∧ ¬A
¬(A ∨ ¬A)
{A, ¬A} ⊨ B (for any B)

The Deduction Theorem

The bridge between object-level → and meta-level ⊨. One of the most important results in propositional logic.

Γ ⊨ A → B
META LEVEL — entailment
The premises in Γ entail the conditional A→B as a conclusion
⟺
IF AND ONLY IF
Γ ∪ {A} ⊨ B
OBJECT LEVEL — adding assumption
Adding A as a premise to Γ entails B directly

→ Left to Right

If Γ ⊨ A→B, then Γ∪{A} ⊨ B
Assume v makes all of Γ∪{A} true. In particular, v makes Γ true. By hypothesis, v makes A→B true. Since v also makes A true, by the truth table for →, v must make B true. ✓
Uses: truth table for → (T→T = T, T→F = F)

← Right to Left

If Γ∪{A} ⊨ B, then Γ ⊨ A→B
Assume v makes all of Γ true. Either v(A)=T or v(A)=F.
Case T: Γ∪{A} all true, so v(B)=T. Then A→B is T.
Case F: A→B is vacuously T regardless of B.
The vacuous case (F→anything=T) is why → is defined this way.

Why This is "Striking"

The philosophical significance
HOW MATHEMATICIANS PROVE CONDITIONALS

To prove "if P then Q," mathematicians universally assume P and then derive Q. The deduction theorem says this strategy is logically valid — assuming A and deriving B is exactly equivalent to proving A→B outright.

WHY → IS DEFINED AS IT IS

The "odd" truth table for → (where F→anything = T) is the only definition that makes the Deduction Theorem hold perfectly. The vacuous truth cases aren't a bug — they're what makes the bridge between ⊨ and → mathematically clean.

Propositional Tableau Rules

Every connective has two rules: one for when it's marked True, one for False. Annotations cite which rule and which line.

How Tableaux Work

TO PROVE Γ ⊨ A

Assume the negation: mark every sentence in Γ as T, and mark A as F. Then apply rules to decompose. If every branch closes (contradiction), the entailment holds.

OPEN BRANCH = COUNTERMODEL

If a branch stays open with no more rules to apply, read off a countermodel directly from the T/F assignments on that branch. The entailment fails.

K

System K

The base modal logic. Named after Saul Kripke. Every other system extends K.

Axiom K — Distribution
◻(A → B) → (◻A → ◻B)
Necessity distributes over implication. If necessarily A implies B, and A is necessary, then B is necessary.
Rule of Necessitation
If ⊢ A, then ⊢ ◻A
If A is a theorem (provable from nothing), then it's necessarily true. Tautologies are necessary.

The Core Intuition

K has no constraints on the accessibility relation. World w can access v, or not — anything goes. ◻A means "A is true in all worlds accessible from here," but there's no guarantee about which worlds those are or whether any exist. K is the minimal commitment logic.

World Structure

No constraints on accessibility. Worlds can relate arbitrarily.

Frame Properties

✗ Reflexive✗ Symmetric✗ Transitive✗ Euclidean

What K Cannot Prove

◻A → A (T axiom)
A → ◻◇A (B axiom)
◻A → ◻◻A (4 axiom)

T

System T

K plus reflexivity. The first system that feels realistic about necessity.

Inherits from K
◻(A → B) → (◻A → ◻B)
All of K's axioms carry over.
Axiom T — Reflexivity
◻A → A
If A is necessary, then A is true. What must be, is. Every world accesses itself.

The Core Intuition

Axiom T adds one constraint: every world can access itself. This is reflexivity. Without T, you could have a world where ◻A holds but A is false — because that world doesn't access itself. T rules that out. This is the minimum for a sensible notion of necessity.

World Structure

Every world accesses itself (reflexive). Represented by self-loops.

Frame Properties

✓ Reflexive✗ Symmetric✗ Transitive✗ Euclidean

Key Insight

T is the weakest system where necessity actually implies truth. Often taken as the baseline for alethic (truth-based) modal logic.

B

System B

K + T + symmetry. If something is actual, it's necessarily possible.

Axiom T (inherited)
◻A → A
Necessity implies truth.
Axiom B — Symmetry
A → ◻◇A
If A is true, then necessarily it's possible that A. What is actual is necessarily possible.

The Core Intuition

Symmetry means: if world w can see world v, then v can see w back. The accessibility relation is two-way. No world is "invisible" to the worlds it can see. If there is a possible world where things are different, then from that world, the actual world is also accessible as a possibility.

World Structure

Symmetric: if w sees v, then v sees w. All arrows are bidirectional.

Frame Properties

✓ Reflexive✓ Symmetric✗ Transitive✗ Euclidean

The B Axiom Plain English

A → ◻◇A: "If it's raining, then necessarily it's possible that it's raining." What actually happens can't be impossible from any world that can see this one.

4

Axiom 4 / System K4

K + transitivity. Necessity of necessity.

Inherits from K
◻(A → B) → (◻A → ◻B)
Distribution of necessity.
Axiom 4 — Transitivity
◻A → ◻◻A
If A is necessary, then it's necessarily necessary. Necessity chains collapse.

The Core Intuition

Transitivity means: if w can see v, and v can see u, then w can see u directly. Chains of accessibility collapse. If A is true in all worlds reachable from w, and those worlds can reach further worlds, then A must be true in those further worlds too — so ◻A implies ◻◻A. Note: K4 does NOT include T — necessary things don't have to actually be true.

World Structure

Transitive: if w→v and v→u, then w→u. Chains collapse (dashed arrow).

Frame Properties

✗ Reflexive✗ Symmetric✓ Transitive✗ Euclidean

Key Point

K4 does NOT include T. ◻A → A doesn't hold. You can have necessity without truth. Add T to K4 and you get S4.

S4

System S4

K + T + 4. Reflexive and transitive. Standard for epistemic and intuitionistic logic.

Axiom T (reflexivity)
◻A → A
Necessary things are true.
Axiom 4 (transitivity)
◻A → ◻◻A
Necessary things are necessarily necessary.

The Core Intuition

S4 combines reflexivity (T) and transitivity (4) but not symmetry. The frame is a preorder. In epistemic logic: if I know A, I know that I know A (positive introspection). But I don't necessarily know what I don't know. S4 also corresponds to intuitionistic logic via the Gödel translation.

World Structure

Reflexive + transitive (preorder). Self-loops and collapsing chains. No back-arrows required.

Frame Properties

✓ Reflexive✗ Symmetric✓ Transitive✗ Euclidean

Famous Application

S4 models knowledge with positive introspection: "I know that I know what I know." Also the basis for intuitionistic logic via the Gödel translation.

S5

System S5

K + T + 4 + B. Equivalence relations. Every world sees every world.

Axiom T
◻A → A
Necessity implies truth.
Axiom 4
◻A → ◻◻A
Necessity of necessity.
Axiom 5 / E
◇A → ◻◇A
If A is possible, it's necessarily possible. All worlds share the same accessibility set.
Net Result
Universal Accessibility
Every world can access every other world. Complete equivalence relation.

The Core Intuition

S5 is the logic of absolute necessity and possibility. Because every world sees every other world, what's possible is the same from every vantage point. ◇A means A is true in some world — period. ◻A means A is true in all worlds — period. No relativity to a particular world's perspective. This is the standard system for metaphysical modality.

World Structure

Universal access: every world sees every world. Complete graph with self-loops.

Frame Properties

✓ Reflexive✓ Symmetric✓ Transitive✓ Euclidean

The Collapse

◻◻A = ◻A, ◇◇A = ◇A, ◻◇A = ◇A, ◇◻A = ◻A. Stacking modal operators collapses. There's only one level of modality.
